# README — Global Ads Performance Dataset

## Overview

This dataset contains campaign-level digital advertising performance data spanning
7 countries, 3 major platforms, 5 industry verticals, and 4 campaign types across
the full calendar year 2024. It is used as the primary dataset for a regression
analysis studying the relationship between digital ad spend and campaign revenue.

- **Filename:** global_ads_performance_dataset.csv
- **Format:** CSV (comma-separated values)
- **Rows:** 1,800
- **Columns:** 14
- **Missing values:** 0
- **Time period:** January 2024 — December 2024
- **License:** Apache 2.0 — permitted for academic and student use
- **Source:** Kaggle — Global Ads Performance Dataset
- **Date downloaded:** March 2026

| Column | Type | Description | Range / Values |

| `impressions` | Integer | Total number of times the ad was shown | 5,059 to 199,650 |
| `clicks` | Integer | Total number of clicks on the ad | 91 to 16,660 |
| `CTR` | Float | Click-through rate (clicks / impressions) | 0.0089 to 0.0956 |
| `CPC` | Float | Cost per click (USD) | $0.28 to $3.95 |
| `ad_spend` | Float | Total amount spent on the campaign (USD) | $58.00 to $38,453.32 |
| `conversions` | Integer | Number of customer conversions achieved | 2 to 1,151 |
| `CPA` | Float | Cost per acquisition (ad_spend / conversions, USD) | — |
| `revenue` | Float | Total revenue generated by the campaign (USD) | $142.69 to $295,028.26 |
| `ROAS` | Float | Return on ad spend (revenue / ad_spend) | 0.13 to 49.0 |

## Usage in This Project

This dataset is used to build a multivariate OLS regression model predicting
campaign revenue (target variable Y) from the following features:

**Continuous predictors:** ad_spend, clicks, impressions, conversions, CTR, CPA

**Categorical controls (dummy encoded):** platform, campaign_type, industry, country

**Time control:** date (extracted to month/quarter for seasonality)

### Correlations with revenue (key findings)
- conversions: +0.83 (strongest predictor)
- clicks: +0.68
- impressions: +0.48
- ad_spend: +0.48
- CTR: +0.41
- CPA: -0.26 (higher cost per acquisition = lower revenue)
